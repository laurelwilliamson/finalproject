About the Project

This website was deployed as part of a group project for a Data Analysis Bootcamp. The project was focused on visualization on the web! Each of of the pages was created using a different data visualization technique.

  • Home: The data is visualized using Leaflet.js, with the Choropleth.js plugin.
  • Scatter Plot: The plot is made using d3.js to draw everything, from the axis, labels, circles, and tooltips! The slider is created using d3-simple-slider.
  • Country Data: Plotly.js is used to make the plot. The table is created using the d3 methods .data(), .enter(), and .append()!

About the Data - The World Happiness Report

The data used for all of our visualizations was found on Kaggle. It originates with the World Happiness Report (WHR).
The data contains information on over 150 countries. In the project, we focused on the Happiness Score for each country, as well as six explanatory factors from the WHR:

  • GDP per Capita
  • Family/Social Support
  • Healthy Life Expectancy
  • Freedom to make Life Choices
  • Perception of Government Corruption
  • Generosity

The datapoint associated with each factor is a measurement to help explain how much of an impact that factor had on the Happiness Score for the country.
